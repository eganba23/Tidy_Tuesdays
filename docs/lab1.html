Exercise 2

Code to create:

dino_data <- datasaurus_dozen %>%
  filter(dataset == "dino")
ggplot(data = dino_data, mapping = aes(x = x, y = y)) +
  geom_point()

Code to calculate the correlation between x and y in this dataset:

dino_data %>%
  summarize(r = cor(x, y))
